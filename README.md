# 💸 Smart Expense Tracker

A modern and responsive **Expense Tracker Web App** built using **HTML, CSS, and JavaScript**.  
This project helps users manage their **income and expenses**, track their **balance**, and store transactions using **LocalStorage**.

## 🚀 Features

- ➕ Add new transactions
- 💰 Track total balance
- 📈 Separate income and expenses
- 🗑️ Delete transactions
- 📅 Transaction date display
- 💾 Data saved in browser using LocalStorage
- 🎨 Modern Glassmorphism UI
